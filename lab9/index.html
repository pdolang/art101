Lab 9: Libraries & jQuery

Challenge

For this lab, we were assigned to create a JavaScript file in Visual Studio Code with jQuery. Within this webpage, we were asked to create a button with a text label for each section of this lab index.html file. Using jQuery, we created a click event for each button. For this webpage, I changed the color of each section whenever the user clicked the button. Clicking the button back will reverse the color back to the original.

Problems

I had a lot of difficulties differentiating which button belonged to each section. I also was very confused about how JavaScript knew what to reference in both the CSS file and the HTML file. Thankfully, the in-class lecture cleared up all my questions about it.

Reflection

I had a lot of fun customizing each button and styling each section. Through this lab, I definitely understand the relationships between JavaScript, HTML, and CSS much more.
