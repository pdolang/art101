Lab 7: Functions

Challenge

For this lab, we had to create a JavaScript file that asks for user input of their name, or whatever characters they inputted. With this input, we created an array by splitting up each individual character. Then, we sorted through the array with an alphabetical organizational method. Finally, we created a string that combined all the sorted characters of the array and neatly printed out the string for the user to see their "new name."
